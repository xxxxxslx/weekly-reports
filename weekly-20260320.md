## FastAPI

### 核心参数类型

| 参数类型 | 定义方式 / 核心标识 | 常用限制 / 校验手段（关键） | 位置 / 传输方式 | 适用场景 |
| :--- | :--- | :--- | :--- | :--- |
| 路径参数 | 1. 基础：`/path/{param}` + 函数参数<br>2. 进阶：`Path(...)` 标识 | - `Path(ge=1, le=100)`：数值范围（≥1/≤100）<br>- `Path(min_length=3)`：字符串最小长度<br>- `Path(regex="^\\d+$")`：正则匹配<br>- 类型约束（int/str/ 枚举） | URL 路径中（如 `/user/123`） | 资源标识（用户 ID、商品 ID）、固定分类 |
| 查询参数 | 1. 基础：函数参数（非路径参数）<br>2. 进阶：`Query(...)` 标识 | - `Query(default=1)`：默认值<br>- `Query(None)`：可选参数<br>- `Query(max_items=10)`：列表最大项数<br>- `Query(alias="page_num")`：参数别名 | URL 后缀（`?page=1&size=10`） | 分页、筛选、排序、搜索关键词 |
| 请求体参数 | 1. Pydantic 模型（`BaseModel`）<br>2. 单个值：`Body(...)` | - Pydantic 字段约束：`Field(gt=0)`<br>- 嵌套模型校验<br>- 必填 / 可选字段（`None` 标识）<br>- 数据类型强制校验（str/int/list） | HTTP 请求体（JSON 格式） | 创建 / 修改资源（提交用户信息、订单） |
| 表单参数 | `Form(...)` 标识（需安装 `python-multipart`） | - `Form(max_length=50)`：字符串长度<br>- `Form(pattern="^\\w+$")`：正则<br>- 必填 / 可选（同 `Query`） | 请求体（表单编码格式） | 登录、表单提交、简单数据录入 |
| 文件参数 | `UploadFile` / `File(...)` 标识 | - `UploadFile` 限制文件类型：`media_type="image/*"`<br>- `File(max_size=1024*1024)`：文件大小（1MB） | 请求体（multipart 格式） | 图片 / 文档上传、批量文件提交 |
| 头部参数 | `Header(...)` 标识 | - `Header(default="zh-CN")`：默认值<br>- `Header(convert_underscores=False)`：保留下划线<br>- 正则 / 长度约束（同 `Query`） | HTTP 请求头中（如 `Token: xxx`） | 鉴权（Token）、语言 / 版本控制、跨域 |
| Cookie 参数 | `Cookie(...)` 标识 | - 同 `Query` 约束（默认值 / 可选 / 正则）<br>- 类型校验（str/int） | HTTP Cookie 中 | 会话保持、用户状态缓存 |

### 响应类型

| 响应类型 | 核心用途 | 核心类/用法 | 关键特点 |
| :--- | :--- | :--- | :--- |
| JSON 响应（默认） | 接口返回结构化数据（最常用） | 直接返回字典/列表/Pydantic 模型；<br>可选 `response_model` 约束结构 | 自动序列化，默认状态码 200；<br>`response_model` 可过滤字段、约束类型 |
| HTML 响应 | 返回网页/HTML 片段 | `from fastapi.responses import HTMLResponse`<br>路由添加 `response_class=HTMLResponse` | 直接返回 HTML 字符串，浏览器会渲染为页面 |
| 重定向响应 | 跳转到其他 URL/接口 | `from fastapi.responses import RedirectResponse`<br>返回 `RedirectResponse(url="目标地址")` | 支持 301（永久）/307（临时）等重定向状态码 |
| 文件响应 | 返回文件（预览/下载） | `from fastapi.responses import FileResponse`<br>传入文件路径 + 可选 `filename`（下载名） | 自动处理文件读取，支持图片/PDF/普通文件；<br>适合中小文件 |
| 流式响应 | 返回大文件/实时数据（视频/日志） | `from fastapi.responses import StreamingResponse`<br>传入生成器/文件流 | 逐块返回数据，不占用大量内存；<br>适合大文件、实时数据流 |
| 自定义响应 | 定制状态码/响应头 | 方式1：返回 `(数据, 状态码, 响应头)`；<br>方式2：注入 `Response` 对象修改 | 灵活设置状态码（如 201/202）、自定义响应头（如 Token/版本号） |
| 异常响应 | 抛出标准化错误信息 | `from fastapi import HTTPException`<br>`raise HTTPException(status_code, detail)` | 自动返回标准化错误 JSON；<br>支持自定义错误详情、响应头 |

### 补充说明
1.  **响应模型（response_model）**：所有响应类型都可搭配 `response_model` 使用，核心作用是「约束返回结构、过滤敏感字段、自动生成文档」；
2.  **媒体类型（media_type）**：FileResponse/StreamingResponse 需指定 `media_type`（如 `image/png`/`video/mp4`），浏览器会根据类型解析数据；
3.  **状态码规范**：
    - 200：成功（默认）
    - 201：创建资源成功（如新增用户/订单）
    - 301/307：重定向
    - 400/401/404：客户端错误
    - 500：服务端错误
